# How to RUN:

to save pixel art to file:
```
poetry run python procedural_pixel_art -s

```

to see pixel art:
```
poetry run python procedural_pixel_art 

```

You might generate 100 pixelarts by one command!:
```
poetry run python procedural_pixel_art -s -n 100

```
